# RoflanUtils
Social Plugin for Bukkit/Paper servers, that could improve player(s) experience on server.

## Legend
- `<>` – Required
- `()` – Optional
- `[]` – Sub Commands

## Commands

#### `/invsee <player>`

#### `/item [rename] <...arg>`

#### `/mute <player> <duration> <reason>`

#### `/report [send/close]`

#### `/[vanish/cloak/v]`
